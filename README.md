# ConfuserEx Discord bot

Discord bot to obfuscate C# Assemblies with ConfuserEx

## Requirements
1. [NodeJS ](https://nodejs.org/en/)
2. [Mono ](https://www.mono-project.com/)

## Installation

Install the packages

```bash
yarn install
```
Edit the .env and add your token to it
